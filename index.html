URL Scheme Test Page

Testing various URL schemes and payment intents

About This Page

This test page demonstrates different URL schemes including:

  • Standard schemes: http, https, file, chrome, data, javascript, about, copy
  • Payment intents: UPI payment links that should trigger payment applications
  • Non-standard schemes: Custom and other protocol handlers

Payment Intent Links (UPI)

upi: UPI Payment Link - ₹100 Should trigger UPI app with payment intent for ₹100
upi: UPI Payment Link - ₹50.50 Should trigger UPI app with payment intent for ₹50.50
upi: UPI Payment Link - ₹250 Should trigger UPI app with payment details

Standard Schemes (Allowed List)

These schemes fall under the standard allowed list: http, https, file, chrome, data, javascript, about, copy

http: HTTP Link Standard HTTP protocol
https: HTTPS Link Secure HTTPS protocol
file: File Link Local file protocol (may not work in browser for security reasons)
chrome: Chrome Settings Chrome internal page (works only in Chrome)
data: Data URI Data URI with inline HTML
javascript: JavaScript URI JavaScript protocol (triggers alert)
about: About Blank About page (opens blank page)
copy: Copy Scheme Copy protocol (may not be widely supported)

Non-Standard Schemes (Not in Allowed List)

These schemes do NOT fall under the standard allowed list

mailto: Email Link Mail protocol (opens email client)
tel: Phone Link Telephone protocol (triggers dialer on mobile)
sms: SMS Link SMS protocol (opens messaging app)
ftp: FTP Link File Transfer Protocol
whatsapp: WhatsApp Link WhatsApp protocol (opens WhatsApp if installed)
slack: Slack Link Slack protocol (opens Slack if installed)
custom: Custom Scheme Custom application protocol
intent: Android Intent Android Intent URI (works only on Android)
market: Play Store Link Android Market/Play Store protocol
tg: Telegram Link Telegram protocol (opens Telegram if installed)
